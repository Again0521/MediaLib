# 视频播放器功能补全计划

> 2026-06-11 整理。对比对象：IINA、mpv、VLC、PotPlayer、Infuse、Movist Pro、Elmedia。
> 结论：MediaLIB 播放器在「媒体库联动」（进度记忆、自动下一集、片头片尾、在线字幕、章节标记、离线缓存）方面已超过多数通用播放器，差距集中在**系统集成**（防休眠、画中画、音频设备）与**画面/字幕的深度自定义**。

## 一、现状盘点（已具备，无需重复建设）

- 播放控制：倍速、快进/快退间隔、A-B 循环、单片循环、逐帧步进（快捷键）、上一集/下一集、剧集列表与自动续播队列（2026-06-11 新增）、自动下一集、恢复播放回退
- 画面：画面比例、黑边裁切、去隔行、旋转、画面均衡器（亮度/对比度/饱和度/伽马/色相）、硬件解码模式
- 字幕：内嵌/外挂轨道切换、同目录自动加载、在线搜索下载（Podnapisi/OpenSubtitles）、字幕延迟、字幕大小、字幕位置
- 音频：音轨切换、音频延迟、变调保护、音量记忆
- 时间轴：进度条预览图（双模式）、章节/手动标记、片头片尾跳过策略、剩余时间
- 交互：自定义快捷键（含冲突让位）、触控板手势（横扫进度/纵扫音量/捏合全屏）、右键菜单、双击全屏、控制栏右键隐藏（2026-06-11 新增）、控制栏锁定、窗口置顶、窗口宽度
- 其它：截图（含模式选择）、播放信息面板、多清晰度切换（远程）、Emby/Plex 回报、离线缓存播放

## 二、差距清单与优先级

### P0 —— 影响基本观影体验（建议下一轮立即做）

| # | 功能 | 参照 | 说明 / 实现路径 |
|---|------|------|----------------|
| 1 | **播放时防休眠/防屏保** | 所有主流播放器 | 当前完全缺失：长片播放中屏幕会变暗休眠。`ProcessInfo.beginActivity(.idleDisplaySleepDisabled)` 或 `IOPMAssertionCreateWithName`，播放开始持锁、暂停/关闭释放 |
| 2 | **音量增强（>100%）** | IINA/VLC/PotPlayer | 很多片源音轨偏小声。mpv `volume-max=200`，音量条 100% 以上分段提示（颜色区分），滚轮/快捷键同步支持 |
| 3 | **字幕样式自定义** | IINA/VLC/PotPlayer | 已有大小/位置，缺字体、颜色、描边、阴影、背景半透明底。mpv `sub-font` / `sub-color` / `sub-border-size` / `sub-back-color`，放入播放器设置「字幕」新分页 |
| 4 | **手动加载任意字幕文件** | 所有主流播放器 | 字幕弹层加「从文件加载…」（NSOpenPanel → mpv `sub-add`），补齐非同目录字幕场景 |

### P1 —— 对齐主流播放器的常用功能

| # | 功能 | 参照 | 说明 / 实现路径 |
|---|------|------|----------------|
| 5 | **二级字幕（双字幕）** | IINA/PotPlayer | mpv `secondary-sid`，双语对照刚需；字幕弹层加「第二字幕」选择组 |
| 6 | **音频输出设备选择** | IINA/VLC | mpv `audio-device-list` 枚举 + `audio-device` 切换，音轨弹层底部加设备组 |
| 7 | **镜像翻转** | VLC/PotPlayer | 旋转已有，补水平/垂直翻转：mpv vf `hflip`/`vflip`，右键菜单「画面旋转」旁 |
| 8 | **视频滤镜：锐化/降噪/去色带** | PotPlayer/mpv | mpv `vf` 链（`unsharp`、`hqdn3d`、`deband`），画面均衡器卡片内追加三个档位开关 |
| 9 | **播放结束行为** | IINA/VLC | 「自动下一集」之外补充：播完关闭窗口 / 停在最后一帧 / 循环，作为设置「播放行为」一项 |
| 10 | **书签（时间点收藏）播放器内闭环** | PotPlayer/VLC | `PlaybackMarker.bookmark` 模型已存在，缺播放器内「添加书签 + 书签列表跳转」UI；并入章节标记弹层 |
| 11 | **URL / 网络串流直接播放** | IINA/VLC/Elmedia | 菜单「打开网络串流…」接受 http(s)/rtmp 等 URL 直接进播放器（不入库），mpv 天然支持 |
| 12 | **长按倍速（按住 2x-3x 速览）** | PotPlayer/移动端习惯 | 长按右方向键（或可配置键）临时倍速、松开恢复；窗口级 keyDown/keyUp 监听已有基础 |

### P2 —— 进阶/差异化（按需排期）

| # | 功能 | 参照 | 说明 |
|---|------|------|------|
| 13 | 画中画 / 迷你悬浮窗 | IINA | libmpv 自渲染路线下原生 PiP 不可行（需 AVSampleBufferDisplayLayer），用「无边框置顶小窗 + 极简控制」模拟 |
| 14 | 每视频倍速/音轨/字幕记忆 | PotPlayer | 现仅默认倍速全局生效；按媒体 ID 记忆最近选择，写 MediaLIB 内部索引 |
| 15 | 视频侧音频 EQ | PotPlayer | 复用音乐 `AudioEQProcessor` 思路，但 libmpv 路线需走 mpv `af` 链（`equalizer`/`superequalizer`） |
| 16 | HDR 色调映射选项 | IINA/mpv | mpv `tone-mapping` 几种曲线选择，HDR 片源观感 |
| 17 | 鼠标手势 / 中键侧键映射 | PotPlayer | 中键播放暂停、侧键前后集等，可并入快捷键体系 |
| 18 | GIF / 连拍截图 | PotPlayer | 截图能力已有，扩展区间导出（调 ffmpeg） |
| 19 | 性能/统计 OSD | mpv stats | 丢帧、码率、缓存时长实时面板，排查弱网有用（播放信息面板已有静态版，可加实时刷新项） |
| 20 | 章节缩略图条 | 自有特色 | 进度条预览已有，章节点上叠缩略图入口 |

## 三、建议实施节奏（每轮结束打包 dmg 验收）

- **R1（P0 一把抓）**：防休眠 + 音量增强 + 字幕样式自定义 + 手动加载字幕。全是 mpv 现成属性/系统 API，风险低、感知强。 ✅ **已完成（2026-06-11）**
- **R2（字幕音频进阶）**：二级字幕 + 音频设备选择 + 镜像翻转 + 锐化/降噪（去色带原已具备）。 ✅ **已完成（2026-06-11）**
- **R3（播放流增强）**：播放结束行为 + 书签闭环 + URL 串流播放。 ✅ **已完成（2026-06-12）**
- **R4（交互差异化）**：长按倍速 + 鼠标手势 + 每视频记忆。 ✅ **已完成（2026-06-12）**
- **R5（大件）**：迷你悬浮窗（PiP 替代）、HDR 色调映射、视频音频 EQ —— 单独评审后再做。

## 四、明确不做 / 暂缓

- **AirPlay 视频投送**：libmpv 渲染到自有 OpenGL 视图，AVRoutePicker 无法搬运画面（控制栏注释已记录此结论），整窗投屏交给 macOS 屏幕镜像。
- **原生系统 PiP**：同上，渲染管线不兼容，仅做置顶小窗模拟（R5）。
- **倍速范围扩到 4x+**：mpv 高倍速下音频同步质量差，维持当前范围。
